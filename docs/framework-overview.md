# AI Disruption Investment Analysis Framework

## Introduction

This framework provides a structured analytical methodology for evaluating technology equities in the AI era. Traditional valuation metrics remain necessary but insufficient — the AI revolution requires an additional layer of disruption risk assessment.

## The Core Insight

In a pre-AI market, investors debated **when** disruption might affect a company's cash flows. The AI revolution has shifted certain sectors into a fundamentally different mode: markets now debate **if** those cash flows will survive at all.

This is a categorical shift in how risk is priced:

- **WHEN mode**: Timing uncertainty, but confidence in durability
- **IF mode**: Existential uncertainty, massive risk premium demanded

Understanding which mode applies to each position is the foundation of AI-era portfolio management.

## Framework Structure

The framework consists of six analytical lenses, each examining a different dimension of risk:

1. **When→If Disruption Classifier** — Categorises market sentiment
2. **Three-Lever Repricing Model** — Decomposes valuation changes
3. **Tactical vs Structural Decomposition** — Distinguishes temporary from permanent
4. **Jevons Paradox Test** — Identifies demand expansion opportunities
5. **Build-vs-Buy Power Shift** — Assesses customer DIY vulnerability
6. **Survival Fitness Assessment** — Evaluates balance sheet resilience

## Application Sequence

### Pre-Market Analysis
1. Scan for AI-related catalysts
2. Update When→If classifications
3. Decompose pre-market moves
4. Flag Jevons opportunities
5. Generate action recommendations

### Post-Market Analysis
1. Review day's price action
2. Update Three-Lever assessments
3. Scan for Build-vs-Buy developments
4. Update Survival Fitness scores
5. Identify contrarian opportunities

## Key Principles

1. **Multiple lenses, triangulated conclusions** — No single lens is sufficient
2. **Classification changes matter more than static classifications** — Track transitions
3. **Tactical opportunities exist within structural concerns** — Degrossing creates entry points
4. **Jevons Paradox is underappreciated** — Markets over-apply disruption narratives
5. **Cash flow durability is the ultimate arbiter** — Survival enables adaptation
